# MASARI BRAND GUIDELINES
# AI-Powered Educational Platform | VARK Learning Framework

## BRAND OVERVIEW
Masari is an AI-powered educational platform that helps students learn through personalized, multi-modal content generation based on the VARK learning framework.

Tagline: "Learn Your Way"
Mission: 4 personalized learning paths for every student

## LOGO SPECIFICATIONS

### Logo Mark
Abstract mark featuring four elegant curved pathways converging into a stylized "M" shape:
- Purple path (Visual)
- Teal path (Audio)
- Orange path (Read/Write)
- Green path (Kinesthetic)

### Logo Usage
- Minimum clear space: Height of the "M" on all sides
- Minimum size: 24px height for digital, 10mm for print
- Always use approved color combinations
- Never distort, rotate, or alter the logo proportions

## PRESENTATION/POWERPOINT GUIDELINES

### Slide Backgrounds
- Light Mode: Cloud White (#F8FAFC) or pure white
- Dark Mode: Deep Navy (#1E293B) gradient

### Slide Structure
- Title slides: Logo centered, dark background with colorful accents
- Content slides: Light background, navy text
- Section dividers: Bold color treatment with brand colors
- Data slides: Clean charts using VARK color palette

### Color Usage in Presentations
- Use all four VARK colors for variety across slides
- Purple for Visual-related content
- Teal for Audio-related content
- Orange for Read/Write-related content
- Green for Interactive/Kinesthetic content
- Navy for all primary text
- Never use more than 3 accent colors per slide

## DESIGN DO'S AND DON'TS

### DO:
✓ Use the four VARK colors consistently
✓ Maintain clean, generous whitespace
✓ Use modern sans-serif typography
✓ Include the logo on all materials
✓ Keep designs student-friendly and approachable
✓ Use gradients subtly between brand colors

### DON'T:
✗ Use colors outside the brand palette
✗ Crowd designs with too many elements
✗ Use serif or decorative fonts
✗ Alter logo colors or proportions
✗ Make designs feel corporate or cold
✗ Mix too many accent colors in one design
